
# Recipe Management System (Django)

A full-stack web application built using Django that allows users to manage recipes with authentication.

## Features

- User Registration and Login
- Add Recipe
- Update Recipe
- Delete Recipe
- Image Upload for Recipes
- Secure Authentication System

## Tech Stack

- Python
- Django
- SQLite
- HTML / CSS / Bootstrap
